About BhaataPhod

    About BhaataPhod

  • I did immense "MaathaPhod" to create "BhaataPhod"
    (baa dum tssh) pun intended! LOL

    How it all started

    As a aspiring Game Developer i was looking for ideas to create a game on for my learning, then i stumbled upon a old atari disk i had in my shelf and thought of the legendary game "Asteroids", i thought i could create a "Asteroids" clone and maybe add a few elements of my own.

    The Implementation

    I coded this game entirely in a single C++ script taking help of SFML 2.6.1 Library and using AI to help me debug this program, coding this game was a hella ride

    All of the pixel artwork used in this game is made by me and yeah i used PisKel for designing them

    The shockwave powerup, UFOs and asset management were the most challenging works of this game
